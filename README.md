# team-project-auth
회원가입/로그인 백엔드 + 프론트 구현

# 🔐 로그인 / 회원가입 기능

## 📌 소개
팀 프로젝트용 회원가입 및 로그인 기능 구현입니다.

## 🛠 기술 스택
- Frontend: React (또는 사용한 것)
- Backend: Node.js / Express
- DB: MySQL (또는 MongoDB)
- Auth: JWT

## ✨ 주요 기능
- 회원가입
- 로그인
- 토큰 기반 인증
- 프론트-백 연동
